cryptoKS.js Tutorial

This guide shows how to use cryptoKS.js to build an offline registration and login system using only HTML and JavaScript.

📦 Step 1: Include cryptoKS

<script src="cryptoKS.min.js"></script>

🔐 Step 2: Register User


await cryptoKS.saveUser({
  name: "John Doe",
  email: "john@example.com",
  password: "secret123"
});

🔓 Step 3: Login from File


await cryptoKS.loginFromFile(user => {
  alert("Welcome, " + user.name);
});

💡 Tips

  • No backend needed
  • 100% offline and secure
  • Works on any browser
